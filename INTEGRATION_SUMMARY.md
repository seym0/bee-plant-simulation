## Структура данных, собираемых логгером

### `logger.years`
- Тип: список целых чисел
- Содержит: номера лет (1, 2, 3, ..., 65)

### `logger.bee_population`
- Тип: список целых чисел
- Содержит: размер популяции пчел на конец каждого года

### `logger.plant_data`
- Тип: словарь `{name: list}`
- Ключи: названия видов растений
- Значения: списки seed_bank по годам

Пример:
```python
{
    'Очень ранний вид': [10000, 9500, 8500, ..., 0],
    'Ранневесенний вид': [10000, 9500, 9100, ..., 7],
    ...
}
```

### `logger.temperature_mean`
- Тип: список вещественных чисел
- Содержит: среднегодовую температуру

### `logger.bee_activation_day`
- Тип: список целых чисел
- Содержит: день в году, когда пчелы просыпаются (1-365)

## Выходные файлы

### simulation_results.png

**Размер:** ~393 KB  
**Разрешение:** 300 DPI  
**Формат:** PNG

**Содержит два подграфика:**

#### Верхний (Популяции)
- X-ось: Годы (1-65)
- Y-ось: Размер популяции
- Данные:
  - Оранжевая толстая линия: популяция пчел
  - Цветные пунктирные линии: seed_bank растений

#### Нижний (Фенологический сдвиг)
- X-ось: Годы (1-65)
- Y-ось слева: День года (50-150)
- Y-ось справа: Температура (°C)
- Данные:
  - Синяя линия (левая ось): день просыпания пчел
  - Красная линия (правая ось): среднегодовая температура
