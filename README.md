# Симулятор рассинхронизации пчел и растений

## Запуск интерактивной визуализации в streamlit
```bash
streamlit run app.py
```

## Настройка окружения
### 1. Клонирование репозитория
Если вы еще этого не сделали:
```bash
git clone https://github.com/seym0/bee-plant-simulation.git

cd bee-plant-simulation
```

### 2. Создание виртуального окружения
```bash
python3 -m venv .venv
```

### 3. Активация окружения
- macOS / Linux:
```bash
source .venv/bin/activate
```

- Windows:
```bash
.venv\Scripts\activate
```

### 4. Установка необходимых библиотек
```bash
pip install -r requirements.txt
```
